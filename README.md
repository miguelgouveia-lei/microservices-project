# Final Project - Cloud Information Systems

Cloud-native microservices project based on the laboratory reference application.

## Architecture

The system includes multiple Java microservices, Docker containers, AWS infrastructure, Terraform automation, Ansible configuration management, SQS asynchronous communication, RDS persistence, and GitHub Actions CI/CD.

## Main Components

- API Gateway
- Product Service
- Order Service
- User Service
- AWS VPC, EC2, RDS, SQS
- Terraform modules
- Ansible playbooks
- GitHub Actions workflows
- DockerHub image registry

## Project Goal

The goal is to demonstrate cloud engineering practices, not complex business logic.
